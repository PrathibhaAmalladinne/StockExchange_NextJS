This is a software built on Next.js to display the last reported financial data for all companies for a stock exchange with an objective that retail investors can pick one, two, multiple or all companies and compare valuable financial metrics.
All the data used in this project is a mock-up created using Chat-GPT for all the metrics Total number of shares, Promotors holdings, Revenue, PAT, EBIDTA, Fixed Assets, Liabilities, Number of employees, Last Dividend per share, Revenue Growth QoQ, Revenue Growth YoY and Last Updated Date. the mock-data used is present in the route.ts folder.
